# SpektrumDX9

Model configurations and system settings for my Spektrum DX9 transmitter (9-channel, DSMX).

## Models

| # | Model | Type |
|---|-------|------|
| 001 | Nano CP S | Micro heli |
| 002 | Nano S3 | Micro heli |
| 003 | Blade 230 Smart | Mid-size heli |
| 004 | AccuRC 2.0 | Simulator |
| 005 | Align 450L Dominator 6S | 450-class heli |

## Files

| File/Directory | Contents |
|----------------|----------|
| `*.SPM` | Model configuration files |
| `MY_DX9.xml` | Transmitter system settings export |
| `SPMTX211.SAX` | DX9 firmware/system archive |
| `Sound_v109/` | Voice pack files (v1.09) for multiple languages |
