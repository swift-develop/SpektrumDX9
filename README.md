# SpektrumDX9

Backup of model configurations and system settings for my Spektrum DX9 transmitter (9-channel, DSMX). Primarily helicopter models.

## Usage

**Backup (SD card → repo):**
Copy the contents of the DX9 SD card into this repo's root directory.

**Restore (repo → SD card):**
Copy the repo contents onto a FAT32-formatted SD card and insert into the DX9.

## SD Card Structure

| Directory | Contents |
|-----------|----------|
| `Aircraft/` | Model configuration files |
| `System/` | Transmitter system settings |
